# configuration.md

To configure NeuralMesh-Pi-Network, you can modify the config.py file in the project directory.

## Here are the available configuration options:

1. UPLOAD_FOLDER: The path to the folder where uploaded images are stored.
2. ALLOWED_EXTENSIONS: The allowed file extensions for uploaded images.
3. MAX_CONTENT_LENGTH: The maximum size of uploaded images in bytes.
4. MODEL_PATH: The path to the trained neural network model.
5. PREDICTION_THRESHOLD: The prediction threshold for generating 3D meshes.
